# Entware Disk Manager 

![Platform](https://img.shields.io/badge/platform-Entware%20%7C%20Keenetic-green)

Интерактивный менеджер дисков и разделов для **Entware / Keenetic** 💻✨  

Позволяет из **(резервного) Entware** управлять разделами USB-дисков, форматировать накопители и создавать резервные копии Entware прямо из терминала через удобное меню.

---

## ⚙️ Возможности

  ### 💾 Управление разделами
  - Создание разделов из **неразмеченного пространства**
  - Создание **нескольких разделов подряд**
  - Удаление существующих разделов 🗑️
  - Форматирование разделов в **ext4** 🆕
  - Полная очистка диска (wipe) 🔥
  
  ### 📊 Информация о дисках
  - Список всех подключённых дисков 🖴
  - Просмотр таблицы разделов 📑
  - Список разделов системы
  - Отображение **UUID и меток** 🔖
  
  ### 🛠 Работа с Entware
  - Создание **бэкапа текущего Entware** (`/opt`) 📦
  - Сохранение архива на любой смонтированный раздел
  - Автоматическое имя архива:
    `Entware_backup_<NAME>_<DATE>.tar.gz`
    
    Пример: `Entware_backup_MyRouter_2026-03-05.tar.gz`
---
  
## 📝 Требования

- Роутер с установленным Entware
- Доступ к терминалу через SSH
- Подключённый USB-диск для операций с разделами

# 🚀 Запуск

Скрипт запускается в среде **Entware**.  
Скопируйте и вставьте команду, чтобы запустить Entware Disk Manager:
```
opkg update && opkg install curl bash && curl -O https://raw.githubusercontent.com/LostGit77/Entware-Disk-Manager/refs/heads/main/entware_disk_manager.sh && bash entware_disk_manager.sh
```

# 🖥 Главное меню
```
========== ENTWARE MANAGER ==========
1) Работа с разделами диска
2) Создать бекап смонтированного Entware
0) Выход
```

- **1)** Работа с разделами диска — открывает меню управления дисками и разделами
- **2)** Создать бекап смонтированного Entware — создаёт архив текущего /opt
- **0)** Выход — завершение работы скрипта


## 💾 Меню работы с разделами
```
1) Создать новые разделы из неразмеченного пространства
2) Удалить существующий раздел
3) Отформатировать раздел
4) Полностью очистить диск
5) Показать информацию о дисках
6) Показать список разделов
7) Показать UUID и метки
8) Показать таблицу разделов выбранного диска
0) Назад
```

  ### 1️⃣ Создание новых разделов
  Используется, если на диске есть свободное (нераспределённое) место.  
  
  **Пошагово:**
  1. Выберите диск по номеру из списка.  
  2. Введите размер нового раздела в MB (например: `1000`).  
  3. Укажите метку раздела (например: `OPKG`).  
  4. Опционально: создать несколько разделов подряд или расширить последний раздел до конца диска.  
  5. После завершения операций можно **перезагрузить роутер**.
  
  
  ### 2️⃣ Удаление раздела 🗑️
  1. Выберите номер раздела для удаления.  
  2. Подтвердите удаление.  
  
  > ⚠️ Все данные на разделе будут **безвозвратно удалены**.
  
  
  ### 3️⃣ Форматирование раздела
  1. Выберите номер раздела.  
  2. Раздел будет полностью отформатирован в **ext4** (рекомендуется для Entware).  
  
  > Можно использовать для очистки конкретного раздела без удаления других.
  
  
  ### 4️⃣ Полная очистка диска 🔥
  1. Выберите диск для полной очистки.  
  2. Подтвердите вводом `YES`.  
  3. Все разделы и таблица разделов будут удалены.  
  4. После очистки скрипт предложит создать новые разделы.
  
  
  ### 5️⃣–8️⃣ Информационные пункты
  - **5)** Показать информацию о дисках  
  - **6)** Показать список разделов системы  
  - **7)** Показать UUID и метки  
  - **8)** Показать таблицу разделов выбранного диска  
  
  > 🔹 Ничего не удаляют — только отображают информацию.

## 📦 Меню создания бэкапа Entware

1. Скрипт создаёт архив текущего `/opt`.
2. Предложит ввести короткое имя для бэкапа.
3. Предложит выбрать, на какой смонтированный раздел сохранить архив.  
   Если ничего не вводить, архив будет сохранён в текущем `/opt`.

> 🔹 После создания рекомендуется перенести файл в место хранения резервных копий.

## 🛠 Развёртывание бекапа Entware
В роутере **Keenetic**:  
1. Создаём папку `install` на отформатированном разделе.  
2. Переносим архив бекапа в эту папку.  
3. В **OPKG** в разделе **Накопители** выбираем наш раздел.
4. Нажимаем **Сохранить**  
5. Система автоматически развернёт бекап.  
6. Все программы и настройки Entware восстановятся.

## Установка OPKG Entware на встроенную память роутера
> 🔹 Резервный Entware

**Для поддержки данной возможности вам потребуется установить пакеты:**
| 1. | Поддержка открытых пакетов              | ✅     |
| 2. | Модули ядра для поддержки файловых систем | ✅     |

**Начиная с KeeneticOS 4.2 появилась возможность установки в одну команду через CLI:**
```
192.168.1.1/a
```
**Для архитектуры `Mipsel`:**
```
opkg disk storage:/ https://bin.entware.net/mipselsf-k3.4/installer/mipsel-installer.tar.gz
```
**Для архитектуры `Aarch`:**
```
opkg disk storage:/ https://bin.entware.net/aarch64-k3.10/installer/aarch64-installer.tar.gz
```
**Дождаться окончания установки в Журнале**
